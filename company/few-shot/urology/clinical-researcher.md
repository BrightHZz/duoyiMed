# Urology Clinical Researcher — Few-Shot Example

## Example: 前列腺癌风险分层 — 临床问题操作化

**用户请求**: "设计一个预测前列腺活检结果的模型，帮助减少不必要的活检"

### Step 1: 定义目标人群
- 年龄: ≥40 岁男性 (有家族史者 ≥40，无家族史者 ≥50)
- 临床场景: PSA 升高 (4-10 ng/mL 灰区) 或 DRE 异常，首次活检
- 排除: 既往活检阳性者、已确诊前列腺癌、5ARIs 使用中（影响 PSA）

### Step 2: 操作化结局变量
- 主要结局: 活检临床显著前列腺癌 (csPCa) — ISUP Grade Group ≥ 2 (Gleason ≥ 3+4)
- 次要结局: 任何前列腺癌 (GG ≥ 1)
- 注意: GG1 (Gleason 3+3) 患者通常适合主动监测，因此 csPCa 是更有临床意义的结局

### Step 3: 候选预测因子池
| 领域 | 变量 |
|------|------|
| 人口学 | 年龄、种族、前列腺癌家族史 |
| 临床 | PSA、游离/总PSA比值 (f/t PSA)、PSA密度 (PSAD = PSA/前列腺体积)、DRE 发现 |
| 影像 | 多参数 MRI: PIRADS v2.1 评分 (1-5)、前列腺体积、可疑病灶数量/位置 |
| 既往史 | 既往阴性活检次数(如适用)、5ARI 使用史 |
| 合并症 | BMI、糖尿病 (与前列腺癌风险负相关) |

### Step 4: 数据可用性映射
- MIMIC-IV: 可获取 PSA、DRE、合并症，但缺少 MRI 影像和系统活检结果
- SEER: 癌症确诊后数据，无筛查阶段信息
- **推荐**: 需合作医院的泌尿外科 EHR + PACS 数据，或使用公开数据集 (如 ProstateX, PROSTATEx)

### Step 5: 临床效用阈值
- 灵敏度 ≥ 0.90 (避免漏诊 csPCa)
- 特异度目标 ≥ 0.40 (减少 40% 不必要的活检)
- Decision Curve: 阈值概率 8-20% 范围内须有净获益
- NNB (Number Needed to Biopsy): 目标 <3 (每发现 1 例 csPCa 需要的活检数)

### 输出: 临床问题定义文档

```markdown
## 临床问题定义 — 前列腺癌活检结果预测

### 目标人群
PSA 4-10 ng/mL 灰区 + 首次活检的男性 (≥40岁)

### 结局定义
- 主要: 临床显著前列腺癌 (ISUP GG ≥ 2)
- 次要: 任何前列腺癌 (GG ≥ 1)

### 候选预测因子 (18 个)
[列表...]

### 数据需求
优先: 合作医院泌尿外科数据库 (PSA + MRI PIRADS + 活检病理)
备选: 公开数据集 (ProstateX)
```
