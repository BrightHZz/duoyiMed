# PI 终审签批标准模板

> 适用所有事业部 (urology / geriatrics / pediatrics)。各事业部可在标准清单基础上追加事业部特有检查项。

---

## 标准终审清单 (7 项)

执行研究项目终审签批时，PI 必须逐项确认以下清单。全部通过方可签批。

- [ ] **1. 前置审查产物完整性**
  - biostatistician 统计审查报告: EXISTS + verdict=approved
  - clinical-researcher 临床审查报告: EXISTS + verdict=approved
  - data provenance 报告 (data-engineer): EXISTS + DAG-R4 验证通过

- [ ] **2. 数值内部一致性**
  - Abstract 数值 = Results 数值 = Tables 数值 = Figures 数值
  - 所有百分比和效应量可在原始分析输出文件中追溯

- [ ] **3. 研究问题-结论闭环**
  - Conclusion 明确回答了 Introduction 中提出的研究问题
  - 无过度推广 (不过度外推到研究人群以外的群体)
  - 事业部的核心贡献增量一句话可表述清楚

- [ ] **4. 局限性充分披露**
  - 事业部特有的局限性已完整列出 (如数据来源限制、随访不完整、选择偏倚)
  - 局限性讨论与 Methods/Results 中的实际限制一致, 非模板化

- [ ] **5. 投稿就绪**
  - 目标期刊与 manuscript 声明一致
  - Cover letter 已就绪
  - 所有参考文献 DOI 已验证 (期刊 DOI ≥95%, 全部 DOI/URL ≥80%)

- [ ] **6. 参考文献全面达标**
  - 数量: 论著 ≥25 篇 / 综述 ≥45 篇 (不足则退回补充)
  - 经典文献占比: ≤ 总参考文献 5% (超标 → 多余经典不再豁免, recency 可能 FAIL)
  - 时效性: 近5年 ≥80% + 近10年 ≥95% (不达标则退回补充近期文献或注册经典豁免)
  - reference-claim-mapping.md 完整性: 双向一致, 孤儿引用=0
  - 来源层级: 全部引用 source_tier ≥ L2, L3=0 (禁止), L2 ≤30% (COND_PASS)
  - 综述禁止引用综述: 论据来自一次文献, 综述仅限 gap identification

- [ ] **7. 引用抽样审计 (综述项目专属)**
  - 从 manuscript 参考文献列表中随机抽取 N = min(5, 10%×refs) 篇
  - 打开每篇原文, 核对最关键的 1 条引用声明
  - 抽样结果 ≥4/5 通过 → 接受剩余风险
  - 抽样结果 <4/5 → MAJOR REVISION, 返回 Phase 3' 全面审计证据表

## 事业部特有检查项 (追加)

各事业部可在上述标准清单基础上追加以下类型的特有检查项:

- 疾病特异性局限性 (如泌尿外科: 结石成分数据缺失)
- 领域特异性统计约束 (如老年医学: 衰弱量表的跨文化验证)
- 目标期刊的特殊投稿要求

追加格式:
```
- [ ] **事业部特有 — [描述]**: [检查内容]
```

---

*此模板为全公司强制执行。各 PI 终审签批必须覆盖标准 7 项。缺失任一项 → 退回补充。*
