# 儿科正常值参考范围

以下范围为健康儿童的近似范围，具体 cut-off 因指南不同有差异。ICU 中需根据临床上下文判断。

## 生命体征 (按年龄)

### 心率 (Heart Rate, beats/min)

| 年龄 | 正常范围 | 平均值 |
|------|---------|--------|
| 新生儿 (0-28d) | 100-180 | 140 |
| 婴儿 (1-12m) | 100-160 | 130 |
| 幼儿 (1-3y) | 90-150 | 120 |
| 学龄前 (3-6y) | 80-140 | 100 |
| 学龄期 (6-12y) | 70-120 | 90 |
| 青少年 (>12y) | 60-100 | 80 |

### 呼吸频率 (Respiratory Rate, breaths/min)

| 年龄 | 正常范围 |
|------|---------|
| 新生儿 | 30-60 |
| 婴儿 (1-12m) | 25-45 |
| 幼儿 (1-3y) | 20-40 |
| 学龄前 (3-6y) | 20-30 |
| 学龄期 (6-12y) | 16-24 |
| 青少年 (>12y) | 12-20 |

### 收缩压 (Systolic BP, mmHg)

| 年龄 | P5 | P50 | P95 |
|------|-----|------|------|
| 新生儿 | 50 | 65 | 80 |
| 婴儿 (1-12m) | 70 | 85 | 100 |
| 幼儿 (1-3y) | 75 | 90 | 105 |
| 学龄前 (3-6y) | 80 | 95 | 110 |
| 学龄期 (6-12y) | 85 | 100 | 120 |
| 青少年 (>12y) | 95 | 110 | 130 |

**快速估算公式**: SBP = 70 + (年龄 × 2) mmHg (1-10 岁)

### 体温 (Temperature)

- 正常: 36.5-37.5°C
- 低热: 37.5-38.0°C
- 发热: >38.0°C
- 高热: >39.0°C
- 新生儿发热需高度重视 (>38.0°C → 有创检查指征)

## 实验室参考值

### 血常规

| 指标 | 新生儿 | 婴儿 | 儿童 | 青少年 |
|------|--------|------|------|--------|
| Hb (g/dL) | 14.5-22.5 → 降至 9-14 (2-3月生理性贫血) | 10-13 | 11-14 | 12-16 (M) / 11.5-15 (F) |
| WBC (×10⁹/L) | 9-30 | 6-17 | 5-13 | 4-10 |
| PLT (×10⁹/L) | 150-400 | 150-400 | 150-400 | 150-400 |

### 血气分析

| 指标 | 参考范围 |
|------|---------|
| pH | 7.35-7.45 |
| pCO₂ (mmHg) | 35-45 (新生儿稍高: 30-45) |
| pO₂ (mmHg) | 80-100 (新生儿: 50-70) |
| HCO₃⁻ (mmol/L) | 22-26 (新生儿稍低: 20-24) |
| BE (mmol/L) | -3 to +3 |
| 乳酸 (mmol/L) | < 2.0 |

### 炎症标志物

| 指标 | 参考范围 | 儿科特殊性 |
|------|---------|-----------|
| CRP (mg/L) | < 5-10 | 新生儿 CRP 升高滞后于感染 12-24h；出生 24-48h 后参考值才稳定 |
| PCT (ng/mL) | < 0.05-0.1 | 新生儿生后 48-72h 有生理性升高，之后降至 < 0.5；细菌感染 > 0.5 |
| ESR (mm/h) | < 10-20 | 随年龄变化 |

### 肾功能

| 指标 | 新生儿 | 婴儿/儿童 | 青少年 |
|------|--------|----------|--------|
| Cr (mg/dL) | 0.3-1.0 (出生前几天反映母体水平) | 0.2-0.4 | 0.5-0.9 |
| BUN (mg/dL) | 3-12 | 5-18 | 7-20 |

**eGFR 估算**: Schwartz 公式 — `eGFR = k × Height(cm) / Cr(mg/dL)`
- k = 0.33 (VLBW <1y), 0.45 (term <1y), 0.55 (child/adol female), 0.70 (adol male)

### 肝功能

| 指标 | 参考范围 |
|------|---------|
| ALT (IU/L) | < 40-50 |
| AST (IU/L) | < 40-60 |
| 总胆红素 (mg/dL) | 新生儿: 随日龄变化（Bhutani 曲线）；>1 月: < 1.0 |
| 直接胆红素 | < 0.2 mg/dL |
| Albumin (g/dL) | 新生儿 2.8-4.4 → 儿童 3.5-5.0 |

## 危急值 (Critical Values)

| 指标 | 危急值 |
|------|--------|
| pH | < 7.20 or > 7.60 |
| pCO₂ | > 70 mmHg |
| pO₂ | < 50 mmHg |
| 乳酸 | > 4.0 mmol/L |
| 血糖 | < 40 mg/dL (新生儿) / < 50 mg/dL (儿童) / > 300 mg/dL |
| 血钾 | < 3.0 or > 6.5 mmol/L |
| 血钠 | < 125 or > 155 mmol/L |
| 血小板 | < 20 ×10⁹/L |
| Hb | < 6 g/dL |

> **注意**: 上述范围为通用近似值，不应替代各医院检验科的参考范围。不同检测方法 (POCT vs 中心实验室) 之间的差异可达 10-15%。
