# PIC 数据库快速参考

> 完整文档见 [[../datasets/pic/PIC|PIC 数据库主页]]

## 基本信息

| 属性 | 值 |
|------|------|
| 全称 | 儿科重症监护数据库 (Pediatric Intensive Care database) |
| 来源 | 浙江大学医学院附属儿童医院 |
| 患者数 | ~12,881 |
| 时间范围 | 2010-2018 |
| 覆盖单元 | PICU, NICU, CCU |
| 数据类型 | EHR (电子健康记录) |

## 三个单元的患者分布 (估计)

| 单元 | 患者数 | 关键人群 | 主要结局 |
|------|--------|---------|----------|
| NICU | ~4,000-5,000 | 早产儿 (VLBW/ELBW)、新生儿外科 | 死亡率 ~5-10%，BPD ~30%，ROP ~15% |
| PICU | ~5,000-6,000 | 儿童（28天-18岁）危重症 | 死亡率 ~3-6%，机械通气率 ~30-40% |
| CCU | ~2,000-3,000 | 先天性心脏病患儿 | 术后死亡率 ~3-5%，并发症 ~20-30% |

## 核心数据表

| 表名 | 内容 | ML 建模作用 |
|------|------|-----------|
| `patients` | 人口学 + 出生信息 | 基础特征（年龄/性别/出生体重/胎龄） |
| `admissions` | 入院/出院/转科 | 结局定义（死亡/住院时间/再入院） |
| `icustays` | ICU 停留详情 | 暴露定义（ICU 时长/入ICU来源） |
| `labevents` | 实验室检查结果 | 特征工程（时序/最差值/趋势） |
| `chartevents` | 床旁监护数据 (生命体征) | 时序特征（HR/RR/BP/SpO₂） |
| `microbiologyevents` | 微生物培养 + 药敏 | 感染结局定义（败血症/耐药） |
| `prescriptions` | 药物处方 | 治疗特征（抗生素/Vasoactives/IVIG） |
| `diagnoses_icd` | ICD 编码诊断 | 合并症、疾病表型 |
| `inputevents` | 静脉输液/用药 | 液体平衡、血管活性药物剂量 |
| `outputevents` | 出量 (尿量/引流) | 肾功能、液体平衡 |
| `surgery_info` | 手术信息 | 手术类型/时长/麻醉 |
| `emr_symptoms` | 入院主诉/症状 | NLP 可用 |

## 特色优势

1. **中国最大的单中心儿科 ICU 数据库** — 涵盖 NICU/PICU/CCU 三个单元
2. **含时序数据** — 生命体征、实验室、用药均有时序信息，可做纵向建模
3. **含手术信息** — CCU 的先天性心脏病手术数据是独特资源
4. **含微生物+药敏** — 可研究儿科感染、抗生素耐药模式

## 局限性

1. **单中心** — 外推性受限，需讨论人群/地域/年代代表性
2. **回顾性** — 缺失数据的模式可能与疾病严重度相关
3. **数据年代** — 2010-2018，治疗实践可能已变化（校准度漂移）
4. **部分表型需 ICD 编码提取** — 准确度依赖编码质量

## 注意事项

- 实验室参考范围依年龄而异，需用年龄特异性阈值
- 新生儿生命体征的"正常"范围与儿童/成人完全不同
- 生长发育指标的解读需胎龄/矫正胎龄校正
- 疾病严重度评分 (PRISM III/PIM-3) 需要年龄特异性参数
