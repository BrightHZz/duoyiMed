## Figure 1. Receiver operating characteristic (ROC) curve.

ROC curve for the PSA-free XGBoost model (n = 2,437) from 5-fold stratified cross-validation. Cross-validated AUC = 0.8510 (95% CI: 0.8286–0.8734). The diagonal dashed line represents random classification (AUC = 0.50). The shaded area represents the 95% confidence interval. The inset box displays the cohort size and outcome event rate (19.0%). The curve was generated from pooled out-of-fold predictions across the five cross-validation folds.
