# 辩论观点陈述: Clinical Researcher

**辩论主题**: 前列腺癌住院不良结局预测 — 建模方法选择 + 统计分析策略 + 协变量筛选
**日期**: 2026-05-10
**确信度标注**: [高/中/低]

---

## 我的核心观点 (按重要性排序)

### 观点 1: 不同意「仅做复合结局」的方案；死亡和 ICU 是本质不同的临床终点 [确信度: 高]

**论据**:
- 从临床角度看，前列腺癌患者住 ICU 的原因与死亡的原因有本质区别:
  - **ICU 入院**: 通常因为急性可逆事件 → 术后出血、败血症、心血管失代偿；这些是可以干预和预防的
  - **30天死亡**: 更多反映终末期疾病 → 广泛转移、恶液质、治疗无效；这些可能超出了急性干预的范围
  - 将它们合并为一个结局会模糊模型的临床解释: "模型在预测什么?"
- 审稿人会问: 如果模型预测高风险，临床医生应该做什么? 对复合结局这个问题无法给出统一答案

**反建议**:
- 我接受 biostatistician 的折中方案: **Primary = 复合结局, 但必须做 secondary analysis 分别预测各组成部分**
- 额外要求: **在 Discussion 中用一个段落专门讨论两个结局组成的预测因子差异**
- 如果两个组成部分的 top-5 SHAP 特征差异很大 → 这是一个重要的临床发现

### 观点 2: 协变量必须有明确的临床先验排序，不依赖纯数据驱动 [确信度: 高]

**论据**:
- LASSO 和 SHAP 只能告诉你"数据中什么重要"，不能告诉你"临床上什么重要"
- 如果 Gleason 和 TNM 不可用 (这是巨大遗憾)，以下临床先验不可违反:
  1. **PSA 应该是 top-1 或 top-2 特征** — 如果 SHAP 显示 PSA 不重要，说明数据或编码有问题
  2. **骨转移必须是 top-3** — 转移状态直接改变预后
  3. **年龄方向必须为正** — 年龄增大不应降低风险 (除非有选择偏倚)
- Phase 5 审查时必须逐条验证这些先验

**推荐**: SAP 中预先列出临床先验效应方向表 (见 Phase 1 表型方案 §4)，作为 Phase 5 Gate 的检查清单。

### 观点 3: 要求进行按年龄和转移状态的分层分析 [确信度: 中]

**论据**:
- 前列腺癌在不同年龄段的生物学行为差异较大:
  - < 65 岁: 更多局部疾病/手术相关
  - ≥ 75 岁: 更多合并症驱动/转移性疾病
- 模型可能在年轻组表现好、老年组表现差 (或相反)
- 分层 AUC 是评估公平性的基本手段

**推荐**: 
- Subgroup AUC by: age (<65, 65-74, ≥75), metastasis (yes/no), admission type (elective/emergency)
- 不是 formal hypothesis testing，是 exploratory

### 观点 4: PSA 的操作化需要临床精确性 [确信度: 高]

**论据**:
- 和 glucose/WBC 不同，PSA 不是常规入院检查，测量时机很重要:
  - 择期前列腺切除术前的 PSA vs 急诊入院时的 PSA — 完全不同的临床含义
  - 正在接受 ADT (雄激素剥夺治疗) 的患者的 PSA 被压制，不能反映肿瘤负荷
- 我们需要在数据提取时区分这些情境

**推荐**:
- 入院前 30 天内的最近一次 PSA 作为主特征
- 如果患者在入院期间做了前列腺切除术，PSA 只取术前值
- 额外特征: PSA 是否在入院时测了 (测量与否本身是临床信息)
- ADT 状态: 如果 prescriptions 中有 leuprolide/goserelin/degarelix → 标记为 ADT 患者 (PSA 解读需谨慎)

### 观点 5: 外部验证仅用 MIMIC-III 不够，讨论中需要展示对真实世界部署的思考 [确信度: 中]

**论据**:
- MIMIC-III 来自同一家医院 (Beth Israel)，时间验证有价值但空间泛化性未评估
- 审稿人会问: "How well would this model perform at another hospital?"
- 这是 limitation，需要主动讨论而不是等人问

**推荐**:
- Phase 4 使用 MIMIC-III 做时间外部验证 (接受这是目前最佳选择)
- Discussion 中用一个段落讨论泛化性: missing population (社区医院、非教学医院、国际人群)
- 明确声明下一步: 多中心外部验证 (如 eICU、其他 MIMIC 衍生数据库)

## 我的推荐方案

### 协变量筛选的临床框架

1. **强制纳入 (不可被 LASSO 移除)**:
   - age
   - PSA (log-transformed)
   - bone_metastasis
   - admission_type (emergency indicator)
   - Charlson Comorbidity Index (或其关键子项)

2. **候选纳入 (LASSO 筛选)**:
   - 其他实验室值 (Hb, WBC, PLT, Cr, Alb, Lactate)
   - race, insurance, marital_status
   - prostatectomy_this_admission
   - 其他合并症

3. **强制排除 (临床无意义)**:
   - 超出本专业知识范围 — 需要 data-engineer 提供最终变量列表后确认

### 临床可解释性标准

| 标准 | 操作 | Gate |
|------|------|------|
| SHAP top-5 中 PSA 或骨转移 ≥ 1 | 若不满足 → 数据质量复核 | Phase 5 |
| 效应方向全部符合临床先验 | 逐条检查 → 不通过的标记为异常 | Phase 5 |
| 分层 AUC 差异 < 0.10 | 差异 ≥ 0.10 → Discussion 讨论原因 | Phase 5 |
| 临床专家能解释 top-10 SHAP 特征 | 若有无法解释的特征 → 复查编码 | Phase 5 |

## 关键风险

| 风险 | 严重度 | 缓解 |
|------|--------|------|
| 缺少 Gleason/TNM 使模型的临床可信度大打折扣 | 高 | Discussion 首段明确声明这是最大的 limitation；强调实验室+人口学特征仍然有预测价值 |
| PSA 在非前列腺癌主诉的入院中可能没有测量 | 中 | 标记"PSA 未测"作为单独类别，不做均值插补 |
| ADT 患者识别困难 (门诊处方不可见) | 中 | 仅用住院期间 prescriptions 识别 ADT，可能低估 |
| 骨转移编码不完整 (可能仅在影像报告中，非 ICD 编码) | 中 | ICD 编码在一定程度上可信，但可能漏掉 |
