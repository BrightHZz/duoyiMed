# 泌尿系统肿瘤分期

## 前列腺癌 TNM (AJCC 8th, 2017)

### T 分期 (原发肿瘤)
| 分期 | 定义 |
|------|------|
| T1 | 不可触及/影像不可见 |
| T1a | TURP 标本 ≤5% 癌 |
| T1b | TURP 标本 >5% 癌 |
| T1c | PSA 升高穿刺发现 |
| T2 | 可触及/影像可见，局限在前列腺内 |
| T2a | ≤ 1/2 单叶 |
| T2b | > 1/2 单叶 |
| T2c | 双叶 |
| T3a | 包膜外侵犯 (单侧或双侧) |
| T3b | 精囊侵犯 |
| T4 | 侵犯相邻器官 (膀胱颈/直肠/盆壁) |

### N 分期 (淋巴结)
- N0: 无区域淋巴结转移
- N1: 区域淋巴结转移

### M 分期 (远处转移)
- M0: 无远处转移
- M1a: 非区域淋巴结
- M1b: 骨转移
- M1c: 其他部位

### ISUP Grade Group (2014)
| Grade Group | Gleason | 风险 |
|-------------|---------|------|
| 1 | 3+3=6 | 最低 |
| 2 | 3+4=7 | 低-中 |
| 3 | 4+3=7 | 中-高 |
| 4 | 4+4=8 | 高 |
| 5 | 9-10 | 最高 |

## 膀胱癌 TNM (AJCC 8th)

### T 分期
| 分期 | 定义 |
|------|------|
| Ta | 非浸润性乳头状癌 |
| Tis | 原位癌 (CIS, "flat tumor") |
| T1 | 侵犯黏膜下层 |
| T2 | 侵犯肌层 (MIBC) |
| T2a | 浅肌层 |
| T2b | 深肌层 |
| T3 | 侵犯膀胱周围组织 |
| T4 | 侵犯相邻器官 |

**关键分界线**: T1 (NMIBC) vs T2 (MIBC) — 治疗策略完全不同
- NMIBC (Ta/T1/CIS): TURBT ± 膀胱灌注 (BCG/化疗)
- MIBC (T2+): 根治性膀胱切除 ± 新辅助化疗

## 肾癌 TNM (AJCC 8th)

### T 分期
| 分期 | 定义 |
|------|------|
| T1a | ≤ 4 cm，局限在肾内 |
| T1b | >4 cm 且 ≤ 7 cm |
| T2a | >7 cm 且 ≤ 10 cm |
| T2b | >10 cm |
| T3a | 肾静脉/肾窦脂肪侵犯 |
| T3b | 膈下下腔静脉癌栓 |
| T3c | 膈上下腔静脉癌栓 |
| T4 | 穿透 Gerota 筋膜 |

### Fuhrman/ISUP 核分级
- Grade 1-4 (ISUP/WHO 2016 分级 1-4)
- 高级别 (Grade 3-4) 预后更差

## 睾丸癌 TNM

### T 分期
- pT1: 局限在睾丸/附睾，无血管侵犯
- pT2: 血管侵犯 或 侵犯鞘膜/睾丸旁组织
- pT3: 精索侵犯
- pT4: 阴囊侵犯

### 血清肿瘤标志物 (S 分期)
| 分期 | LDH | hCG (mIU/mL) | AFP (ng/mL) |
|------|-----|-------------|------------|
| S0 | 正常 | 正常 | 正常 |
| S1 | <1.5×ULN | <5000 | <1000 |
| S2 | 1.5-10×ULN | 5000-50000 | 1000-10000 |
| S3 | >10×ULN | >50000 | >10000 |

### IGCCCG 预后分组 (转移性)
- **Good**: ~90% 5 年生存
- **Intermediate**: ~80%
- **Poor**: ~50%

## 对 ML 建模的意义

1. **TNM 作为核心特征**: 所有肿瘤预测模型必须纳入 TNM 分期
2. **分期分层报告**: 模型性能必须按分期分层 (至少分 T1-2 vs T3-4)
3. **分期迁移 (Stage Migration)**: 影像技术进步 → 更早期检出 → 模型校准度可能漂移
4. **Gleason → Grade Group 转换**: 2014 年后文献统一使用 ISUP Grade Group
5. **SEER 数据特点**: SEER 使用 Collaborative Staging / Derived AJCC，需注意版本兼容性
