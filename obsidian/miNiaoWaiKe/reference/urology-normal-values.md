# 泌尿外科常用参考值

## 前列腺相关

### PSA (Prostate-Specific Antigen)
| 年龄 | 正常上限 |
|------|---------|
| 40-49 | < 2.5 ng/mL |
| 50-59 | < 3.5 ng/mL |
| 60-69 | < 4.5 ng/mL |
| 70-79 | < 6.5 ng/mL |

**PSA 密度 (PSAD)** = PSA / 前列腺体积
- < 0.15: 低风险
- > 0.15: 灰区；> 0.20: 需进一步评估

**PSA 速率** (PSA velocity): > 0.75 ng/mL/年 → 警惕

**游离/总 PSA 比值** (f/t PSA): < 0.10 → 高风险 (灰区 4-10 ng/mL 时有用)

### 前列腺体积
- 正常: < 30 mL
- 增大: 30-80 mL
- 显著增大: > 80 mL

## 肾功能

### 血肌酐 (Serum Creatinine)
| 性别 | 正常范围 |
|------|---------|
| 男性 | 0.7-1.3 mg/dL (62-115 μmol/L) |
| 女性 | 0.5-1.0 mg/dL (44-88 μmol/L) |

### eGFR (CKD-EPI 公式)
- ≥ 90: 正常
- 60-89: 轻度下降
- 30-59: 中度下降 (CKD 3 期)
- 15-29: 重度下降 (CKD 4 期)
- < 15: 肾衰竭 (CKD 5 期)

### 尿液分析
| 项目 | 正常值 |
|------|--------|
| 尿比重 | 1.003-1.030 |
| 尿 pH | 4.5-8.0 (平均 6.0) |
| 尿蛋白 | < 150 mg/24h；试纸阴性 |
| 尿 RBC | < 3/HPF |
| 尿 WBC | < 5/HPF |
| 尿培养 | < 10⁵ CFU/mL (清洁中段尿) |

### 24h 尿液 (结石评估)
| 项目 | 正常范围 |
|------|---------|
| 尿量 | 2.0-2.5 L (结石预防目标) |
| 钙 | < 250 mg/24h (男) / < 200 mg/24h (女) |
| 草酸盐 | < 45 mg/24h |
| 柠檬酸盐 | > 320 mg/24h (男) / > 450 mg/24h (女) |
| 尿酸 | < 800 mg/24h (男) / < 750 mg/24h (女) |
| 磷 | 600-1200 mg/24h |
| 镁 | 70-200 mg/24h |
| 钠 | 100-260 mmol/24h |
