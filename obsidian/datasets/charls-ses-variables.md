### 收入/财富 (Household Income 文件)

Household Income 文件列前缀结构:
- `ga*` (599列): 个人收入来源明细
- `ha*` (736列): 家庭资产/支出/转移支付

### 社会经济 SES 复合指标构建

```python
# 5维 SES 指标
ses_vars = {
    'edu': '教育年限 (zba001)',
    'urban': '城乡 (zbd001, 1=城市)',
    'pc_income': '人均收入 = sum(ge010_*) / 家庭规模',
    'pc_expenditure': '人均支出 = ha072 / 家庭规模', 
    'hh_assets': '家庭资产 (ha067)',
}

# 构建方法
# 1. 收入变量 log 转换 (处理右偏)
# 2. 中位数插补缺失值
# 3. Z-score 标准化后取均值
```

## 已知限制

1. **收入变量含大量0值**: 农村老年人群收入来源有限，需 log(1+x) 转换
2. **家庭规模变量缺失率高**: `ha001` 仅 2,166 有效值
3. **缺乏吸烟/饮酒变量**: 尚未在 Health_Status 中找到对应变量名
4. **社会参与变量待提取**: CHARLS 问卷中有社交活动模块 (da056 系列)
5. **跨 wave 变量名一致性**: 不同 wave 的 SES 变量名基本一致，可直接复用映射
