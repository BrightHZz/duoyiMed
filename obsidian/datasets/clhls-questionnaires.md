## 问卷类型

### 1. 存活老人调查问卷 (Survivors Questionnaire)

面向 2017-2018 调查期内存活的受访者，涵盖以下模块:

| 模块 | 变量区组 | 内容 |
|------|:---:|------|
| A. 基本信息 | a1-a2 | 性别、出生日期、民族、居住地 |
| B. 心理/自评 | b1-b48 | 自评健康、生活质量、情绪、焦虑、乐观 |
| C. 认知功能 | c11-c67 | MMSE 30项完整版 (定向力、记忆力、计算力、语言) |
| D. 生活方式 | d1-d92 | 吸烟史、饮酒史、体力活动、饮食频率 |
| E. 功能状态 | e1-e14 | ADL (6项) + IADL (8项) |
| F. 社会/经济/照护 | f1-f149 | 教育、婚姻、居住安排、主要照护者、社会保障、医疗服务利用 |
| G. 健康/体格 | g15-g123 | 15种慢性病自报、血压、心率、心律、BMI、腰围 |

### 2. 死亡老人调查问卷 (Deceased Questionnaire)

面向 2017-2018 调查期内去世的受访者，由**家属/知情人**代为回答:

- 死亡日期与地点
- 临终前健康状况
- 临终前 ADL 功能状态
- 临终前医疗服务利用
- 死因（家属报告）

> 注意: 死亡问卷的 ADL 信息来自家属回顾，可能与临床评估存在差异。

## 问卷使用建议

### 变量对应分析
- 中文问卷 → 查看问题措辞和选项 → 对应 codebook 变量名
- English 问卷 → 对照验证翻译准确性 → 可用于英文论文方法部分引用

### 研究设计时参考
1. 明确变量是在存活问卷还是死亡问卷中
2. 确认目标变量的跳转逻辑（哪些子样本回答了该问题）
3. 代访标注: 认知障碍者可代访，需在分析中标注

### 老年医学常用模块

| 研究目标 | 问卷模块 | 关键变量 |
|------|:---:|------|
| 衰弱评估 | E, G | e1-e14, g101-g1021, g4c1-g4c3 |
| 认知障碍 | C | c11-c67 (MMSE) |
| 慢性病共病 | G | g15a1-g15t1 (15 diseases) |
| 照护需求 | E, F | e1-e14, f5, f111-f113 |
| 心理健康 | B | b31-b39 (抑郁), b41-b48 (焦虑) |
| 健康行为 | D | d1-d92 (吸烟/饮酒/饮食/锻炼) |
| 临终关怀 | 死亡问卷 | 死亡地点、临终 ADL、医疗服务 |
