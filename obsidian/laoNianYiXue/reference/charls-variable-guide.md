# CHARLS 变量速查 (ML 建模用)

> 完整文档见 `obsidian/datasets/charls.md`

## 基本信息

| 属性 | 值 |
|------|------|
| 全称 | 中国健康与养老追踪调查 (China Health and Retirement Longitudinal Study) |
| 基线 | 2011 (Wave 1), N≈17,708 |
| 随访 | 2013 (W2), 2015 (W3), 2018 (W4), 2021 (W5) |
| 人群 | ≥45 岁 (含配偶) |
| 调查内容 | 人口学+健康+功能+认知+经济+社交+血液 |
| 血样 | W1 和 W3 (2011/2015) |

## 按 ML 建模模块的变量速查

## CHARLS 纵向分析常用策略

| 研究设计 | Wave 选择 | 说明 |
|----------|----------|------|
| 预测模型 | W1 基线 → W3 结局 (4 年) | 最常用；有血检 |
| 时序预测 | W3 → W4 → W5 (2-3 年间隔) | 无 W3+ 血检 |
| 轨迹建模 | W1→W2→W3→W4→W5 (10 年) | 衰弱/认知/功能轨迹 |
| 差分分析 | Δ(W3-W1) 预测 W4/W5 结局 | 早期变化预测远期结局 |

## 关键数据清洗提醒

1. **代理受访**: CHARLS 有 proxy respondents → 主观变量受影响 (CES-D/认知)
2. **血检权重**: W3 血检 2015 年检测 → 使用匹配的 cross-section 权重
3. **失访**: W1→W5 累积失访 ≥30% → 建议 inverse probability weighting (IPW)
4. **死亡竞争**: 老年队列中死亡是结局的竞争事件 → Fine-Gray 或 cause-specific
5. **城乡差异**: 所有模型必须校正 `bb002` (城乡) 或分层分析
