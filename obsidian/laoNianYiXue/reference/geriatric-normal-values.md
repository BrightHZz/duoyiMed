# 老年人常用参考值与评估工具

## 体格测量

### 握力 (Handgrip Strength)
| 性别 | 肌少症 cut-off (EWGSOP2) | CHARLS 参考 (60-69y) |
|------|------------------------|---------------------|
| 男 | < 27 kg | ~30-35 kg (均值) |
| 女 | < 16 kg | ~20-25 kg (均值) |

### 步速 (Gait Speed, 4m)
| 分类 | 速度 |
|------|------|
| 正常 | > 1.0 m/s |
| 减慢 | 0.8-1.0 m/s |
| 肌少症/衰弱风险 | ≤ 0.8 m/s |
| 不能完成 | 重度功能受限 |

### 五次椅子站立 (5-Time Chair Stand)
| 分类 | 时间 |
|------|------|
| 正常 | < 11-12 秒 |
| 减退 | 12-17 秒 |
| 功能受限 | > 17 秒 或不能完成 |

### BMI (老年人)
| 分类 | BMI |
|------|-----|
| 低体重 (营养不良风险) | < 22 |
| 正常 | 22-27 (老年人偏高更有利!) |
| 超重 | 27-30 |
| 肥胖 | > 30 |

**肥胖悖论**: BMI 25-30 的老年人死亡率最低（不同于年轻人）。

### 血压 (老年人)
| 分类 | 诊室血压 |
|------|---------|
| 理想 (老年人) | SBP < 130-140 (视虚弱程度) |
| 高血压 | SBP ≥ 140 或 DBP ≥ 90 |
| 体位性低血压 | SBP 下降 ≥ 20 或 DBP 下降 ≥ 10 (站立 1-3 min) |

## 实验室参考值 (老年特异性)

### 血液学
| 指标 | 老年人参考范围 | vs 成人 |
|------|-------------|--------|
| Hb | 男 12.0-15.0 g/dL；女 11.5-14.5 g/dL (略低) | 正常下限略低 |
| WBC | 4.0-10.0 ×10⁹/L (类似) | — |
| 淋巴细胞 | > 1.0 ×10⁹/L (<1.0 = 免疫衰老信号) | — |
| PLT | 150-400 ×10⁹/L (类似) | — |

### 肾功能 (eGFR)
| 年龄 | 正常 eGFR 范围 (mL/min/1.73m²) |
|------|-----------------------------|
| 40-49 | 90-116 |
| 50-59 | 80-110 |
| 60-69 | 70-100 |
| 70+ | 60-95 |

**注意**: eGFR ≥ 60 且无蛋白尿/血尿对老年人可能是正常，不一定是 CKD。

### 营养/炎症
| 指标 | 正常 | 风险 |
|------|------|------|
| 白蛋白 (Alb) | 3.5-5.0 g/dL | < 3.5 = 营养不良/炎症 (老年人常见) |
| 总蛋白 | 6.5-8.0 g/dL | — |
| 维生素 D (25-OH) | ≥ 30 ng/mL | < 20 = 缺乏 (老年人高发) |
| 维生素 B12 | ≥ 200 pg/mL | < 200 = 缺乏 |
| CRP | < 5 mg/L | 5-10 = 低度炎症；>10 = 活动炎症 |
| IL-6 | < 5 pg/mL | 随年龄升高 ("inflammaging") |
| 同型半胱氨酸 | < 15 μmol/L | 随年龄升高；B12/叶酸缺乏相关 |

### 内分泌
| 指标 | 老年人特点 |
|------|----------|
| TSH | 随年龄略升高 (上限可至 6-8 mIU/L 在 80+岁) |
| 空腹血糖 | 同成人标准 (< 100 正常；100-125 糖尿病前期) |
| HbA1c | 老年人目标更宽松 (< 7.5-8.0% 对于虚弱老人) |
| 睾酮 (男) | 随年龄下降 (迟发性性腺功能减退) |

## 身体功能评估 (Short Physical Performance Battery, SPPB)

| 项目 | 4 分 | 3 分 | 2 分 | 1 分 | 0 分 |
|------|------|------|------|------|------|
| 平衡 (三姿) | 并脚 10s + 半串联 10s + 串联 10s | 串联 3-9.9s | 半串联 10s | 半串联 <10s | 不能做 |
| 步速 (4m) | <4.1s | 4.1-4.8s | 4.9-6.2s | 6.3-8.7s | >8.7s/不能 |
| 椅子站立 (5次) | <11.2s | 11.2-13.7s | 13.8-16.7s | >16.7s | >60s/不能 |

**总分**: 0-12
- 0-3: 重度功能受限
- 4-6: 中度
- 7-9: 轻度
- 10-12: 正常

## 营养评估 (MNA-SF)

| 项目 | 评分 |
|------|------|
| 过去 3 月进食减少 | 0 (严重) - 2 (无) |
| 过去 3 月体重下降 | 0 (>3 kg) - 3 (无) |
| 活动能力 | 0 (卧床) - 2 (户外) |
| 过去 3 月急性疾病/应激 | 0 (是) - 2 (否) |
| 神经心理问题 (痴呆/抑郁) | 0 (重度) - 2 (无) |
| BMI | 0 (<19) - 3 (≥23) |
| **总分** | **0-14** |

0-7: 营养不良；8-11: 风险；12-14: 正常
