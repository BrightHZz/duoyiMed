# 老年综合评估 (CGA) 工具集

老年综合评估 (Comprehensive Geriatric Assessment, CGA) 是老年医学的核心方法论：多维度评估 + 多学科团队 + 个体化管理。

## CGA 评估维度

```
                    ┌────────────┐
                    │  CGA 核心  │
                    └─────┬──────┘
         ┌────────┬───────┼───────┬────────┬────────┐
         ▼        ▼       ▼       ▼        ▼        ▼
      医学评估  功能评估  认知评估  心理评估  社会评估  环境评估
```

## 一、衰弱评估工具

### Fried Frailty Phenotype (CHS Criteria)
| 标准 | 操作化 (CHS 原版) | CHARLS 适配 |
|------|----------------|------------|
| 1. 体重下降 | 过去 1 年非自愿下降 >10 磅 (4.5kg) | 自报体重下降 ≥5kg 或 BMI<18.5 |
| 2. 疲乏 (Exhaustion) | CES-D 2 题: 感觉做什么都费力/提不起劲 (≥3d/week) | CES-D 10 题版相同条目 |
| 3. 低体力活动 | 每周消耗 <383 kcal (男) / <270 kcal (女) | IPAQ 简版或自报活动频率 |
| 4. 步速减慢 | 15 英尺 (4.57m) 步行时间 ≥ 性别+身高 cut-off | 无法完成或 >7s |
| 5. 握力下降 | 握力 < BMI 性别分层 cut-off | <28kg (男) / <18kg (女) 或最低 20% |

**分级**:
- 0 条: 健壮 (Robust)
- 1-2 条: 衰弱前期 (Pre-frail)
- ≥3 条: 衰弱 (Frail)

### FRAIL Scale (临床简易版)
| 项目 | 内容 |
|------|------|
| Fatigue | 疲乏 |
| Resistance | 上一层楼困难 |
| Ambulation | 走一个街区困难 |
| Illness | >5 种疾病 |
| Loss of weight | >5% |

≥3 分 = 衰弱；1-2 分 = 衰弱前期

### Frailty Index (FI, Rockwood 缺陷累积模型)
- **原理**: 缺陷数量 / 评估条目总数
- **常用 cut-off**: >0.25 = 衰弱
- **CHARLS 构建 FI**: 可从 30-70 个缺陷条目 (症状/疾病/功能/实验室) 构建
- **优势**: 连续变量，适合 ML；不依赖特定 cut-off

## 三、跌倒风险评估

### TUG (Timed Up & Go Test)
从椅子上站起 → 走 3 米 → 转身 → 走回 → 坐下
- < 10 秒: 正常
- 10-20 秒: 平衡略差 (跌倒风险↑)
- > 20 秒: 行动障碍 (跌到高风险)
- > 30 秒: 需辅助工具

### BBS (Berg Balance Scale)
- 14 项平衡任务，每项 0-4 分，总分 56
- < 45 = 跌倒高风险

### 其他跌倒风险因素
- 过去 1 年跌倒史 (最强预测因子)
- ≥ 4 种药物 (尤其精神活性/降压/降糖)
- 视力下降 (视力表 < 20/40)
- 体位性低血压 (SBP ↓ 20 mmHg)
- 足部问题 (趾甲/鞋不合适)

## 四、抑郁评估

### GDS-15 (Geriatric Depression Scale, 简版)
15 个是否题，>5 分提示抑郁，>10 分提示重度
CHARLS 使用 CES-D 10 题版替代:
- 8 个负性情绪 + 2 个正性情绪条目
- **总分 0-30** (CHARLS 版本)
- ≥ 10: 抑郁症状 (常用 cut-off)
- ≥ 12: 可能重症抑郁

## 五、CGA 在 ML 建模中的应用

1. **CGA 工具提供标准化特征**: 握力/步速/SPPB 等客观测量替代主观报告
2. **多维度特征融合**: 医学+功能+认知+心理+社会，多模态输入
3. **CGA 作为黄金标准**: 预测模型 vs CGA 综合判断对比
4. **纵向 CGA 轨迹**: 每 1-2 年重复评估 → 时序模型

## CHARLS 中可用的 CGA 相关变量

| 维度 | CHARLS 变量 |
|------|-----------|
| 功能 | ADL (6 项), IADL (5 项) |
| 体能 | 握力, 步速 (2.5m), 椅子站立 (5 次), 平衡测试 |
| 认知 | TICS 改编版, 绘图, 自报记忆 |
| 心理 | CES-D 10 |
| 营养 | BMI, 腰围, 自报体重变化 |
| 社会 | 婚姻, 居住, 社交活动, 经济 |
| 疾病 | 自报 14 种慢性病 |
| 感官 | 自报视力/听力 |
| 跌倒 | 过去 2 年跌倒/骨折史 |
