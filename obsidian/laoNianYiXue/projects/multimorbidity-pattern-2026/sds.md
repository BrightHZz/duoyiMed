## 1. 项目概述

### 1.1 研究标题 (草案)
"基于 CHARLS 数据的中国老年人多病共存模式识别与不良结局预测模型构建"

### 1.2 核心研究问题
中国社区老年人的多病共存模式 (multimorbidity patterns) 有哪些可识别的聚类亚型？不同共病模式的聚类亚型能否预测 4 年不良健康结局 (死亡、ADL 失能、住院)？

### 1.3 研究设计类型
**预测模型构建** — 无监督学习 (聚类) + 监督学习 (预测模型)

### 1.4 一句话核心增量
首次在中国全国代表性老年队列中，结合潜在类别分析 (LCA) 识别多病共存聚类亚型，并验证不同亚型对远期不良结局的差异化预测能力——为基于共病模式的老年风险分层提供循证工具。

## 3. 数据源

### 3.1 主数据源: CHARLS

| 属性 | 说明 |
|------|------|
| 全称 | 中国健康与养老追踪调查 |
| 基线 | 2011 (Wave 1), ~17,000 人 |
| 随访 | 2013/2015/2018/2020 (每 2 年) |
| 年龄 | ≥45 岁 |
| 本地路径 | `~/Documents/trae_projects/related to Sarcopenia/charls/` |
| 数据格式 | 原始 Stata `.dta` + 分析就绪 CSV (77 个文件) |

### 3.2 关键变量可用性
- **14 种自报慢性病**: 高血压、糖尿病、心脏病、卒中、慢阻肺、肝病、肾病、消化系统疾病、精神/记忆障碍、关节炎/风湿病、哮喘、恶性肿瘤、帕金森病、甲状腺疾病
- **人口学**: 年龄、性别、教育、城乡、婚姻
- **功能**: ADL (6项)、IADL (5项)
- **认知**: MMSE 类评估、词语回忆
- **抑郁**: CES-D-10
- **体格**: BMI、握力、步速、血压
- **实验室** (部分 waves): CRP、HbA1c、血脂、肌酐、胱抑素 C
- **结局**: 死亡 (退出访谈)、ADL 失能、住院

### 3.3 分析 Wave 选择
- **基线 (聚类识别)**: Wave 1 (2011) — 最大样本量, 14 种慢病数据完整
- **预测窗口**: Wave 1 → Wave 3 (2015), 4 年随访
- **外部时段验证**: Wave 2 (2013) → Wave 4 (2018)

## 4. 研究设计

### 4.1 设计框架
```
Stage 1 — 无监督学习: 多病共存模式识别
  ├── 输入: Wave 1 的 14 种慢性病 (二值变量)
  ├── 方法: 潜在类别分析 (LCA) + K-prototypes (稳健性)
  ├── 输出: 聚类亚型 (3-6 类) + 各亚型特征描述
  └── 验证: Bootstrap 稳定性 + 临床可解释性审查

Stage 2 — 监督学习: 不良结局预测
  ├── 输入: 基线特征 (人口学+共病模式+功能+认知+生活方式)
  ├── 结局: 4 年死亡 / ADL 失能 / 住院
  ├── 方法: Logistic (基线) + XGBoost (主模型)
  ├── 关键对比: 共病模式标签 vs 简单共病计数 (CCI) 的增量预测价值
  └── 验证: 10-fold nested CV + 时间验证 (Wave 2→4)
```

### 4.2 纳入/排除标准
**纳入**:
- CHARLS Wave 1 参与者
- 年龄 ≥45 岁
- 至少报告 1 种慢性病 (多病共存的纳入标准：≥2 种)

**排除**:
- Wave 1 时已失访 (无基线数据)
- 14 种慢病全部缺失 (无法评估共病)

### 4.3 结局定义
| 结局 | 定义 | 数据来源 |
|------|------|---------|
| 全因死亡 | Wave 3 退出访谈确认死亡 | Exit Interview |
| ADL 失能 | ADL 6 项中 ≥1 项需要帮助 (Wave 3) | Health Status |
| 住院 | Wave 1-3 期间 ≥1 次住院 | Health Care |

## 5. 方法学计划

### 5.1 Stage 1: 共病模式聚类
- **主方法**: 潜在类别分析 (LCA)
  - 二值显变量: 14 种慢病 (有/无)
  - 模型选择: BIC + LMR-LRT + Bootstrap LRT
  - 类数范围: 2-8 类
- **稳健性检验**: K-prototypes (混合变量)
- **亚型命名**: 基于条件概率 >0.5 的疾病组合 (如"心血管代谢型"、"呼吸-关节型")
- **可视化**: 条件概率热力图 + t-SNE/UMAP 降维投影

### 5.2 Stage 2: 预测模型
- **基线模型**: Logistic 回归 (年龄+性别+CCI 计数)
- **主模型**: XGBoost (人口学+生活方式+功能+认知+共病模式标签+CCI)
- **增量价值评估**: ΔAUC (LCA 标签 vs CCI 计数), NRI, IDI
- **特征重要性**: SHAP (全局 + 个体)
- **亚组分析**: 按年龄组 (45-64 / 65-74 / ≥75)、性别、城乡分层

### 5.3 验证策略
- 10-fold nested CV (内部验证)
- 时间验证: Wave 2 → Wave 4 (外部时段)
- Bootstrap 95% CI for AUC
- 校准曲线 + Brier Score

### 5.4 统计分析计划 (SAP) 要点
- 缺失数据: 多重插补 (MICE), 报告缺失率
- 正态性检验: Shapiro-Wilk / Kolmogorov-Smirnov
- 多重比较: Bonferroni 校正
- 软件: R 4.x (poLCA, xgboost, caret, MICE) + Python 3.x (scikit-learn, SHAP)

## 6. 关键创新点

1. **首次在中国老年人中系统识别多病共存聚类亚型**: 现有研究多为欧美人群 (SEER-Medicare、UK Biobank)，中国人群因不同的疾病谱 (高卒中、低肥胖) 可能有独特的共病模式
2. **从聚类到预测的完整工作流**: 不仅识别模式，更验证其预测价值——回答"知道共病模式是否比简单计数更有用"
3. **增量预测价值的方法学评估**: 通过 ΔAUC + NRI + IDI 三重指标量化共病模式标签的增值贡献
4. **临床可操作性**: 聚类亚型对应不同的风险分层和干预优先级，可直接指导老年共病管理

### 7.3 推断预测

| 指标 | 预测值 | 置信度 | 依据 |
|------|--------|--------|------|
| 预计总耗时 | 40-60 h | low (无 RunLog) | 含聚类+预测两个 stage |
| 预计成功率 | 中等 | low | LCA 方法成熟，但中文核心需注意学术规范 |
| 瓶颈阶段 | Phase 3 (执行) | medium | LCA 模型选择 + SHAP 解释 + 增量价值检验 |
| 建议期刊 Tier | Tier 3 (中文核心) | high | 无外部数据源验证，建议中文核心稳妥起步 |
| 主要风险 | 共病聚类临床可解释性不足 | medium | LCA 结果需要临床审查把关 |

### 7.4 风险因素
- LCA 聚类数选择可能存在不确定性 → 多指标联合判断 (BIC + LMR-LRT + 临床可解释性)
- 中文核心期刊对方法学复杂度接受度有限 → 主文稿以临床叙事为主，方法细节放补充材料
- 聚类→预测的两阶段设计可能被审稿人质疑过拟合 → 严格分离训练/测试集，使用 nested CV
