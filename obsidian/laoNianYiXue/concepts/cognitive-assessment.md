# 老年认知评估量表

## 1. MMSE (Mini-Mental State Examination)

### 基本信息
- **满分**: 30 分
- **耗时**: 5-10 分钟
- **评估域**: 定向力 (10) + 即刻记忆 (3) + 注意与计算 (5) + 延迟回忆 (3) + 语言 (9)

### 分界值 (教育校正)
| 教育程度 | 正常 | MCI | 痴呆 |
|----------|------|-----|------|
| 文盲 | ≥18 | 14-17 | <14 |
| 小学 | ≥21 | 17-20 | <17 |
| 初中及以上 | ≥25 | 22-24 | <22 |

### 局限性
- 对 MCI 敏感性低 (~50-60%)
- 天花板效应 (高教育者)
- 对执行功能评估不足
- 受文化/教育影响大
- 有版权限制 (PEI)

## 2. MoCA (Montreal Cognitive Assessment)

### 基本信息
- **满分**: 30 分
- **耗时**: 10-15 分钟
- **评估域**: 视空间/执行 (5) + 命名 (3) + 记忆 (5) + 注意 (6) + 语言 (3) + 抽象 (2) + 延迟回忆 (5) + 定向 (6)

### 分界值
| 版本 | MCI 筛查 | 痴呆筛查 |
|------|---------|---------|
| MoCA 原版 | <26 (教育 ≤12年 +1) | <22 |
| MoCA-B (基础版) | <25 (文盲/低教育) | <20 |

### 优势 vs MMSE
- MCI 灵敏度更高 (~80-90% vs ~50-60%)
- 包含执行功能评估 (连线/TMT-B 替代)
- 免费使用 (需培训认证，2020 年后)

## 3. CDR (Clinical Dementia Rating)

### 分级
| 分数 | 分级 | 含义 |
|------|------|------|
| 0 | 正常 | |
| 0.5 | 可疑痴呆 (相当于 MCI) | |
| 1 | 轻度痴呆 | |
| 2 | 中度痴呆 | |
| 3 | 重度痴呆 | |

### CDR-SB (Sum of Boxes)
- 6 个 domain 分数加和：0-18
- **连续变量**，适合 ML 建模

## 4. 常用神经心理学测验

| 测验 | 评估域 | 耗时 |
|------|--------|------|
| TMT-A (连线测验 A) | 注意/处理速度 | 2-3 min |
| TMT-B (连线测验 B) | 执行功能/任务切换 | 3-5 min |
| 词语流畅性 (VF) | 语义记忆/语言 | 1-2 min |
| 动物命名 (1 min) | 语义流畅性 | 1 min |
| BNT (Boston Naming) | 命名 | 10-15 min |
| AVLT/HVLT | 词语学习与记忆 | 15-20 min |
| Clock Drawing | 视空间/执行 | 2-3 min |
| 数字广度 (Digit Span) | 工作记忆 | 3-5 min |
| GDS (老年抑郁量表) | 情绪 (排除假性痴呆) | 5 min |

## 5. CHARLS 中的认知评估

CHARLS 使用了简化版认知评估（非 MMSE）：
- **TICS** (Telephone Interview for Cognitive Status) 改编版
- **评估域**: 定向、记忆 (即时+延迟)、计算 (100-7 序列)、绘图
- **总分范围**: 0-35 (CHARLS 版本)
- **MCI 分界**: 不同研究使用不同 cut-off，常用低于均值 1-1.5 SD
- **局限性**: 不适用于临床痴呆诊断

## 6. HRS 中的认知评估 (类似 CHARLS)

HRS 核心认知评估使用 TICS-m (modified TICS)：
- **总分**: 0-27
- **MCI 分界**: <12 (常用定义，需校正年龄/教育)
- **痴呆概率**: Langa-Weir 算法 (self/proxy + cognitive score)

## 对 ML 建模的指导

1. **认知分数作为连续变量**优于分类 cut-off（减少信息损失）
2. **教育程度必须校正**：认知分数与教育年数强相关 (r=0.3-0.5)
3. **纵向变化 (ΔMMSE/ΔMoCA)** 比单次测量有更高预测价值
4. **CHARLS 的认知数据**：原始条目分数 + 教育校正 z-score 两种方式
5. **主观记忆抱怨 (SMC)**：单独自报"记忆力比同龄人差"也是一个有效特征
